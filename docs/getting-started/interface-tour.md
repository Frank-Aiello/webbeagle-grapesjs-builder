## Left Panel — Blocks

The blocks panel shows available components. Drag any block onto the canvas.

**Categories:**
- **Layout** — Sections, containers, grids
- **Typography** — Headers, paragraphs, badges
- **Content** — Cards, testimonials, stats
- **Forms** — Inputs, textareas, buttons, selects, labels

## Right Panel — Style Manager

Select any element on the canvas to view style controls:

- **Typography** — Font, size, weight, color, alignment
- **Dimension** — Width, height, margin, padding
- **Background** — Color, gradient, image
- **Border** — Width, style, color, radius
- **Position** — Static, relative, absolute, fixed

## Right Panel — Animation (🎬 Animate)

**Click the 🎬 Animate button to open.** This panel controls scroll-triggered animations.

- **ON/OFF toggle** — Enable/disable animation on selected element
- **Preset grid** — fade-up, fade-in, slide-left, slide-right, zoom-in
- **Stagger Delay** — Time between child animations (ms)
- **Duration** — Animation length (ms)
- **Initial Opacity** — Starting opacity (0 = invisible, 1 = fully visible)
- **Trigger** — On Scroll or On Page Load
- **✨ Stagger All Children** — Auto-tag children for staggered animation

## Canvas

The canvas is a GrapesJS iframe. You can:

- Click to select elements
- Double-click text to edit inline
- Drag blocks from the left panel
- Right-click for context menu (copy, delete, etc.)
- Use the device switcher (Desktop / Tablet / Mobile)
